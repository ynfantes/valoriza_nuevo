Integra WhatsApp a tu sistema Valoriza2

« Utiliza la tecnología de WhatsApp para optimizar tus tareas como administrador »

Solo necesitas tu WhatsApp de siempre!!

¿Cómo Funciona?

Es un software externo que se integra a Valoriza2, diseñado para enviar mensajes en forma masiva y automática, directamente desde su sistema de administración de condominio de manera fácil y cómoda.

El software utiliza la tecnología de WhatsApp Web y automatiza el envío de mensajes a los propietarios, con WhatsApp, registrados en Valoriza2.

  1. Vaya el menú: Condominio / Enviar WhatsApp.
  2. Complete los datos requeridos.
  3. Presione el botón Ver WS, para previsualizar los mensajes antes de enviar.
  4. Presione el botón Enviar WS. Se le abrirá automáticamente una ventana de WhatsApp Web.
  5. Escanee el código QR para iniciar sesión.
  6. Observe como se envían los mensajes.

Desde Valoriza2 podrá utilizar las plantillas para enviar mensajes de cobranza, avisos de facturación, pero también podrá enviar mensajes personalizados utilizando todas las variables disponibles. Igualmente tendrá la opción de adjuntar archivos.

¡Envía notificaciones de cobro, avisos de facturación, comunicaciones, archivos adjuntos, mensajes personalizados y más!

Requisitos

Si usted usa WhatsApp como herramienta de trabajo, seguro cuenta con estos requisitos:

  1. Línea celular con WhatsApp o WhatsApp Business.
  2. Destinatarios registrados en la lista de contactos.
    Puedes hacer la carga masiva con Valoriza2
  3. Sistema Valoriza2.

Precio

Mensual: 12$/Mes o los Bs/Mes. equivalentes a la tasa del día.

**Mensajes ilimitados.

Consejos para el uso

El sistema viene integrado con características antibloqueo, como ejemplo: retraso entre envío de mensajes y retraso en la escritura de cada mensaje para simular la transcripción de una persona.

El uso de envío masivo vía WhatsApp puede ser considerado SPAM, por lo cual te recomendamos seguir estos consejos para evitar el bloqueo de tu número o supensión temporal de Whtasapp.

  • Registrar los contactos a los cuales les vas a enviar mensajes.
  • Redactar mensajes de tal forma que no parezcan robotizados, usa términos de tu lenguaje diario.
  • Redactar mensajes, que inviten al destinatario a darte una respuesta. La interacción, diálogo o conversación es señal de uso normal para WhatsApp.
  • No envíes una cantidad muy alta de mensajes por día.
  • Haz uso racional y eficaz de la herramienta.
